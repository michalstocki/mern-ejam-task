# mern-ejam-task

Little playground project on MERN stack

Make sure you have [Node.js](http://nodejs.org/) and [Yarn](https://yarnpkg.com/) installed.

## Running app

1. Install dependencies

```sh
yarn
```

2. Compile code

```sh
yarn run compile

```

3. Run server

```sh
yarn run start
```

4. Open browser at [http://localhost:5000/](http://localhost:5000/)

## Running E2E tests

```sh
yarn run compile
yarn run test-e2e
```

## Checking code style

```sh
yarn run check
```
